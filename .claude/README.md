# Local Project Directory

This `.claude` directory contains project-specific files for this repository.

## Structure

- `plans/` - Project plans and specifications
- `tasks/` - Individual task files
- `specs/` - Technical specifications
- `state/` - Orchestrator state for this project
- `todos/` - Project-specific TODO tracking
- `events/` - Event logs for this project
- `archive/` - Archived completed work

## Usage

All `/plan`, `/orchestrate`, and `/start-task` commands work with this local directory.

## Cleanup

Run `/cleanup` to archive completed work and reset for new planning.
